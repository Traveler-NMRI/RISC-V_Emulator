# RISC-V_Emulator
RISC-V instruction set emulator on Windows.

Now supports 21 instructions of RV32I instruction set(03-19-25).

lui rd,imm | auipc rd,imm | addi rd,rs1,imm | slti rd,rs1,imm | sltiu rd,rs1,imm

xori rd,rs1,imm | ori rd,rs1,imm | andi rd,rs1,imm | slli rd,rs1,imm | srli rd,rs1,imm

srai rd,rs1,imm | add rd,rs1,rs2 | sub rd,rs1,rs2 | sll rd,rs1,rs2 | slt rd,rs1,rs2

sltu rd,rs1,rs2 | xor rd,rs1,rs2 | srl rd,rs1,rs2 | sra rd,rs1,rs2 | or rd,rs1,rs2

and rd,rs1,rs2
